## Definitions And Objects
- Principle of least action: among trajectories with fixed endpoints, the true trajectory makes the action stationary; in the lecture this is usually phrased as “minimized.”
- History / trajectory: the full collection of generalized coordinates as functions of time, \(\{q_i(t)\}\).
- Local law: a differential law such as \(F=ma\) that advances the motion from nearby data \((q_i,\dot q_i)\).
- Global law: a law stated in terms of the whole trajectory through the action.
- Action: \(A[q]=\int L\,dt\), a functional of the full path.
- Lagrangian: \(L(q,\dot q,t)\); for the mechanical systems in this lecture, \(L=T-U\).
- Variation functions: \(f_i(t)\), arbitrary except for vanishing at the endpoints.
- True path notation: \(\hat q_i(t)\) denotes the actual trajectory.
- Canonical momentum conjugate to \(q_i\): \(\pi_i=\partial L/\partial \dot q_i\).
- Generalized force: \(\partial L/\partial q_i\).
- Translation invariance: the action or Lagrangian is unchanged when all positions are shifted by the same constant.
- Central force: a potential that depends only on distance from the origin, \(U=U(r)\).
- Coordinate independence claim: the least-action principle does not depend on which coordinates are used to describe the same physical system.
- Energy remark, explicitly deferred: \(T+U\) is the conserved energy, not \(L=T-U\); the lecture does not derive that conservation law here.

## Derivation Steps
**Integration by parts**
1. Start from the definite integral of a derivative: \(\int_{t_1}^{t_2}\dot F\,dt=F(t_2)-F(t_1)\).
2. Set \(F=f\,g\).
3. Use the product rule: \(\dot F=\dot f\,g+f\,\dot g\).
4. Integrate both terms over \([t_1,t_2]\).
5. Rearrange to isolate \(\int \dot f\,g\,dt\).
6. In the endpoint-vanishing case, drop \([fg]_{t_1}^{t_2}\).

**Arbitrary-function lemma**
1. Assume \(\int_{t_1}^{t_2}a(t)f(t)\,dt=0\) for every allowed \(f(t)\).
2. Choose \(f(t)\) to be nonzero only in a tiny neighborhood of one point.
3. The integral then isolates the value of \(a(t)\) near that point.
4. Since the point was arbitrary, conclude \(a(t)=0\) everywhere on the interval.

**Euler–Lagrange from stationary action**
1. Write the varied path as \(q_i(t)=\hat q_i(t)+\alpha f_i(t)\).
2. Impose fixed endpoints: \(f_i(t_1)=f_i(t_2)=0\).
3. Define \(A(\alpha)=\int_{t_1}^{t_2}L(q(\alpha),\dot q(\alpha),t)\,dt\).
4. Stationarity of the true path gives \(\left.dA/d\alpha\right|_{\alpha=0}=0\).
5. Use \(\partial q_i/\partial\alpha=f_i\) and \(\partial \dot q_i/\partial\alpha=\dot f_i\).
6. Expand \(dA/d\alpha\) by the chain rule into a \(q_i\) term and a \(\dot q_i\) term.
7. Integrate the \(\dot f_i\) term by parts.
8. Drop the boundary term because each \(f_i\) vanishes at \(t_1,t_2\).
9. Factor out the arbitrary \(f_i(t)\).
10. Apply the arbitrary-function lemma to conclude \(\frac{d}{dt}(\partial L/\partial \dot q_i)=\partial L/\partial q_i\).

**Single-particle Newtonian recovery**
1. Insert \(L=\frac12 m\dot x^2-U(x)\).
2. Compute \(\partial L/\partial \dot x=m\dot x\).
3. Compute \(\partial L/\partial x=-\,dU/dx\).
4. Apply Euler–Lagrange.
5. Obtain \(m\ddot x=-\,dU/dx\).

**Many-particle Cartesian recovery**
1. Write \(L=\sum_i \frac12 m_i\dot x_i^2-U(x_1,\dots,x_N)\).
2. For each coordinate \(x_i\), compute \(p_i=\partial L/\partial \dot x_i=m_i\dot x_i\).
3. Compute \(\partial L/\partial x_i=-\partial U/\partial x_i\).
4. Apply Euler–Lagrange coordinate by coordinate.
5. Get \(dp_i/dt=-\partial U/\partial x_i\) for each particle and direction.

**Translation symmetry for two particles on a line**
1. Assume \(U=U(x_1-x_2)\).
2. Introduce \(d=x_1-x_2\).
3. Use the chain rule to get \(\partial U/\partial x_1=dU/dd\).
4. Use the chain rule to get \(\partial U/\partial x_2=-\,dU/dd\).
5. Therefore \(\partial U/\partial x_2=-\,\partial U/\partial x_1\).
6. Write the two Euler–Lagrange equations for \(x_1\) and \(x_2\).
7. Add them.
8. The force terms cancel.
9. Conclude \(d(p_1+p_2)/dt=0\).

**Near-Earth \(x,y\) example**
1. Use \(T=\frac12 m\dot x^2+\frac12 m\dot y^2\) and \(U=mgy\).
2. Form \(L=T-U=\frac12 m\dot x^2+\frac12 m\dot y^2-mgy\).
3. Since \(L\) does not depend on \(x\), Euler–Lagrange gives \(d(m\dot x)/dt=0\).
4. Since \(\partial U/\partial y=mg\), Euler–Lagrange gives \(d(m\dot y)/dt=-mg\).
5. Interpret the result as one conserved momentum component and one non-conserved component.

**Polar-coordinate kinetic energy and angular momentum**
1. Decompose the velocity into a radial piece and a perpendicular piece.
2. Use \(v_r=\dot r\).
3. Use \(v_\perp=r\dot\theta\).
4. Build \(T=\frac12 m(\dot r^2+r^2\dot\theta^2)\).
5. For a central potential, set \(U=U(r)\), so \(L\) has no explicit \(\theta\)-dependence.
6. Compute \(\partial L/\partial \dot\theta=mr^2\dot\theta\).
7. Apply Euler–Lagrange to \(\theta\).
8. Obtain \(d(mr^2\dot\theta)/dt=0\).
9. Rewrite as \(mr^2\dot\theta=\) constant and \(\dot\theta\propto 1/r^2\).

## Notation Choices
- Use \(L\) as the main symbol for the Lagrangian in the notes.
- Use \(\mathcal L\) only when reproducing or discussing the board-facing frame equations directly.
- Use \(A[q]\) for the action.
- Use \(q_i\) for generalized coordinates; use \(x,y,z\) only for Cartesian examples; use \(r,\theta\) for polar coordinates.
- Use \(\hat q_i(t)\) for the true trajectory and \(f_i(t)\) for the variation shape.
- Use \(\alpha\) as a constant deformation parameter, not a time-dependent quantity.
- Use overdot for total time derivative: \(\dot q_i=dq_i/dt\), \(\dot f_i=df_i/dt\).
- Use \(d/dt\) for total time derivative operators acting on composite quantities.
- Use \(\partial/\partial q_i\) and \(\partial/\partial \dot q_i\) for partial derivatives of \(L\).
- Use explicit sums \(\sum_i\); do not rely on Einstein summation in the final notes.
- Normalize the canonical momentum notation to lowercase \(\pi_i\), not uppercase \(\Pi_i\).
- Use \(p_i\) for ordinary Cartesian momentum components in the particle examples.
- Normalize the potential-energy notation to uppercase \(U\), even where the transcript ASR suggests lowercase \(u\).
- In the two-particle chain-rule derivation, use \(d=x_1-x_2\) only as a temporary auxiliary symbol, not as a permanent notation for the chapter.
- Use endpoint notation \([fg]_{t_1}^{t_2}\) for boundary terms.
- Keep the lecture’s phrasing “kinetic minus potential” attached to \(L\); do not silently replace it with Hamiltonian language.
